范围

使用 .form-range 创建自定义 <input type="range"> 控件。 轨道（背景）和拇指（值）的样式都在浏览器中显示相同。 由于只有 Firefox 支持从拇指的左侧或右侧“填充”其轨迹以直观地指示进度，因此我们目前不支持它。

<label for="customRange1" class="form-label">示例</label>
<input type="range" class="form-range" id="customRange1">
禁用

在输入上添加 disabled 布尔属性，使其外观变灰并删除指针事件。

<label for="disabledRange" class="form-label">禁用示例</label>
<input type="range" class="form-range" id="disabledRange" disabled>
最小值和最大值

范围输入具有 min 和 max 的隐含值，分别为 0 和 100。 您可以为那些使用 min 和 max 属性的值指定新值。

<label for="customRange2" class="form-label">示例</label>
<input type="range" class="form-range" min="0" max="5" id="customRange2">
步长

默认情况下，范围输入“捕捉”到整数值。 要更改此设置，您可以指定步长值。 在下面的示例中，我们使用 step="0.5" 将步数加倍。

<label for="customRange3" class="form-label">示例</label>
<input type="range" class="form-range" min="0" max="5" step="0.5" id="customRange3">
